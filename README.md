#Bike Sales Interactive Performance Dashboard (Excel)

## Project Overview
This end-to-end data analytics project explores a retail dataset of 1,000 customers to identify the key demographic drivers behind bicycle purchasing decisions. The final delivery is a centralized, interactive Excel dashboard designed to help stakeholders segment customers and optimize marketing campaigns.

## Key Insights Discovered
* **Income Thresholds:** Customers who bought bicycles had a higher average income (~$57,962) than non-buyers (~$54,874), with the gap being most visible among male consumers.
* **Commute Distance Impact:** The highest conversion rates occur within a short commute window of **0–1 miles** (200 buyers). Purchasing drops off sharply once a commute exceeds 10 miles.
* **Age Demographics:** The **Middle Age** cohort represents the core target market, driving the vast majority of successful purchases.

## Technical Workflow & Skills Applied
1. **Data Cleaning & QA:** Standardized single-character fields (`M/S` to `Married/Single` and `M/F` to `Male/Female`) to resolve raw data inconsistencies.
2. **Feature Engineering:** Used nested conditional logic (`IF` statements) to transform numerical ages into distinct demographic cohorts (*Adolescent, Middle Age, Old*).
3. **Data Aggregation:** Constructed multi-variable Pivot Tables to isolate correlations between income, commute habits, region, and conversion rates.
4. **Dashboard Design & BI:** Built an interactive reporting layout featuring Clustered Column and Line charts, integrated with **Dynamic Slicers** (Region, Education, Marital Status) for real-time stakeholder filtering.

## Project Deliverables
* `Bike sales dashboard.xlsx`: The complete workbook containing raw data, working sheets, pivot tables, and the interactive dashboard.
